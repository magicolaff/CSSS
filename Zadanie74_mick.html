Kolor tła Jednozadaniowy opis:

selektor { background-color: kolor; }

Przykład:
body { background-color: lightblue; }
Tło obrazkowe Jednozadaniowy opis:

selektor { background-image: url(adres); }

Przykład:
background-image: url("file:///c:/CSS/_rozwiazanie/tla_marginesy/grafiki/tlo_1.jpg");
Powtarzanie tła Jednozadaniowy opis:

selektor { background-repeat: typ; }

Przykład:
background-repeat: no-repeat;

Możliwe wartości: repeat, repeat-x, repeat-y, no-repeat
Pozycja tła Jednozadaniowy opis:

selektor { background-position: pozycja; }

Przykład:
background-position: center top;
Zaczepienie tła Jednozadaniowy opis:

selektor { background-attachment: typ; }

Przykład:
background-attachment: fixed;

Możliwe wartości: scroll, fixed, local

Gradient
liniowy		 Jednozadaniowy opis:

Przykład:
background-image: linear-gradient(to right, red, yellow);
